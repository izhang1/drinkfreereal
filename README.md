# Drink Free

Tracking mobile application allowing alcoholics to see how long they last had a drink. This allows a way to provide continous feedback to users who want to stop drinking and start living life more. 
 
# Features

1. Login function that integrates with multiple access points. 
2. Daily tracker for users to show updates on how many days has passed since their last drink
3. Cost tracker, a way to show how much money as an average the user has saved
4. Statistics to show the ill affects of alcohol and problems people face when abusing it


# Features to Come

- Track your friend/family : Allow users support group to be integrated into the same application. Ability to give encourgement, updates, and even feedback on how they are doing.
